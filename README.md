# dotfiles
backup dotfiles

use oh-my-zsh as base, should config on-my-zsh first.

## useage
```
# git clone dotfiles_repo
# add to ~/.bashrc or ~/.zshrc
source /home/qinshulei/dotfiles/init.sh
```
## 目录结构
+ bin    脚手架脚本
+ shell  别名，bash函数定义，会被source
+ vendor 从网络下载额外的命令行工具
+ service 启动常用开机服务
